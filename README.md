## Description
NetPractice is a simple project to learn about networking. The goal is to fix broken network setups so they work correctly. There are 10 levels, each with a picture of a network. You change settings like IP addresses and masks to make the network connect properly.

## Instructions
1. Download the project files from the 42 page.
2. Extract the files into a folder.
3. Run the `run.sh` script to start a web server and open the page in your browser.
4. Enter your login or use "evaluation" for random setups.
5. For each level, change the unshaded fields to fix the network.
6. Click "Check again" to test if it's right.
7. When done, click "Get my config" to download your settings.
8. For submission, put the 10 config files (level1.json to level10.json) in the root of your Git repository.

## Resources
This project covers basic networking ideas like TCP/IP addressing, subnet masks, default gateways, routers, switches, and OSI layers. I learned most by asking peers, trying to understand myself, and remembering some subnetting from 5 years ago. For more info, check simple tutorials on TCP/IP basics. AI was used a little to explain simple concepts.
